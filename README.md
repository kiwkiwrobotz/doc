## 1. Penjelasan Akreditasi

### Apa itu Akreditasi Sekolah?

Akreditasi sekolah adalah proses penilaian kelayakan satuan pendidikan berdasarkan kriteria yang ditetapkan oleh **Badan Akreditasi Nasional Sekolah/Madrasah (BAN-S/M)**. Akreditasi bertujuan untuk:

- âœ… Menjamin mutu pendidikan nasional
- âœ… Memberikan informasi kepada masyarakat tentang kualitas sekolah
- âœ… Meningkatkan kepercayaan stakeholder (orang tua, pemerintah, mitra)
- âœ… Menjadi dasar pembinaan dan pengembangan sekolah

### Tingkatan Akreditasi

| Peringkat | Nilai | Keterangan |
|-----------|-------|------------|
| **A** | 91-100 | Unggul |
| **B** | 81-90 | Baik |
| **C** | 71-80 | Cukup |
| **Tidak Terakreditasi** | < 71 | Perlu perbaikan signifikan |

### Instrumen Akreditasi 2024 (IA2024)

Akreditasi terbaru menggunakan **IA2024** yang merupakan penyempurnaan dari IASP 2020, diterbitkan berdasarkan **Kepmendikbudristek Nomor 246/O/2024**. IA2024 dilaksanakan oleh **BAN-PDM** (gabungan BAN PAUD PNF dan BAN-SM).

IA2024 menilai **4 komponen utama** yang berdampak langsung pada mutu pendidikan:

| No | Komponen | Fokus Penilaian |
|----|----------|-----------------|
| 1 | **Kinerja Pendidik dalam Pembelajaran** | Interaksi, strategi pengajaran, dan penilaian |
| 2 | **Kepemimpinan Kepala Sekolah** | Kemampuan memimpin perbaikan berkelanjutan |
| 3 | **Iklim Lingkungan Belajar** | Lingkungan aman, inklusif, dan kaya keberagaman |
| 4 | **Hasil Belajar Murid** | Pencapaian kompetensi akademik dan karakter |

### Metode Pengumpulan Data IA2024

- ğŸ“„ **Telaah Dokumen** - Bukti administratif dan perencanaan
- ğŸ‘€ **Observasi** - Lingkungan sekolah dan proses pembelajaran

## 4. Cara Menggunakan Aplikasi untuk Akreditasi

### Langkah 1: Persiapan Data Dasar

1. **Masukkan master data** melalui menu **Admin > Pengaturan Sistem**
   - Data sekolah (nama, NPSN, alamat)
   - Tahun ajaran aktif
   - Struktur kelas dan rombel

2. **Import data siswa** melalui **Kesiswaan > Database Siswa**
   - Gunakan template Excel yang disediakan
   - Pastikan NIS/NISN terisi lengkap

3. **Input data guru** melalui **HRIS > Data Pegawai**
   - Lengkapi kualifikasi pendidikan
   - Upload sertifikat pendidik

### Langkah 2: Operasional Harian

Lakukan kegiatan operasional seperti biasa:

| Kegiatan | Menu | Frekuensi |
|----------|------|-----------|
| Presensi siswa | Akademik > Presensi | Harian |
| Presensi guru | HRIS > Absensi | Harian |
| Input nilai | Akademik > Penilaian | Per Topik/UTS/UAS |
| Upload karya siswa | Kesiswaan > Portofolio | Berkala |
| Catat pembayaran | Keuangan > SPP | Sesuai jadwal |
| Update inventaris | Inventaris > Data Barang | Berkala |

### Langkah 3: Generate Laporan Akreditasi

1. Buka menu **Admin > Rencana Kerja (RKT)**
2. Pilih tab **Akreditasi**
3. Sistem akan menampilkan:
   - **Checklist kelengkapan** per standar
   - **Persentase kesiapan** secara visual
   - **Tombol unduh** untuk setiap dokumen

4. Klik **"Generate Paket Akreditasi"** untuk menghasilkan:
   - Folder terstruktur sesuai 8 standar
   - Dokumen dalam format PDF siap cetak
   - Daftar isi dan indeks dokumen

### Langkah 4: Visitasi & Tindak Lanjut

Saat visitasi akreditasi:
- Siapkan **akses readonly** untuk asesor
- Tampilkan **dashboard realtime** sebagai bukti penggunaan sistem
- Berikan **print-out** dokumen pendukung

Setelah visitasi:
- Simpan hasil rekomendasi di sistem
- Buat **rencana tindak lanjut** di menu RKT
- Monitor implementasi perbaikan
